## 🚀 Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-portfolio.git
   ```
2. Navigate to the project folder:
   ```bash
   cd your-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## 📦 Deployment
1. Build the project:
   ```bash
   npm run build
   ```
2. Deploy to GitHub Pages:
   ```bash
   npm run deploy
   ```
